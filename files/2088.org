#+TITLE: Rook/Ceph v1.17 から v1.18 へのアップグレード手順（Helm版）

* 基本手順

下記の手順で実行します。
今回対象のバージョンに対応して実行した内容は後述します。

** helmでデプロイされたバージョンの確認

#+begin_src bash
helm list -n rook-ceph
#+end_src

** バージョン確認

#+begin_src bash
kubectl -n rook-ceph get CephCluster rook-ceph -o jsonpath='{.status.ceph.versions}' | jq .
#+end_src

** クラスター HEALTH の確認

#+begin_src bash
kubectl -n rook-ceph get cephcluster
#+end_src

** rook-ceph-operator podの確認

#+begin_src bash
kubectl -n rook-ceph get pods -l app=rook-ceph-operator
#+end_src

** Deploymentのバージョン確認(単発)

#+begin_src bash
kubectl -n rook-ceph get deployments \
  -l rook_cluster=rook-ceph \
  -o jsonpath='{range .items[*]}{.metadata.name}{"  req/upd/avl: "}{.spec.replicas}{"/"}{.status.updatedReplicas}{"/"}{.status.readyReplicas}{"  rook-version="}{.metadata.labels.rook-version}{"\n"}{end}'
#+end_src

** Deploymentのバージョン確認(watch)

#+begin_src shell
watch --exec kubectl -n rook-ceph get deployments \
  -l rook_cluster=rook-ceph \
  -o jsonpath='{range .items[*]}{.metadata.name}{"  req/upd/avl: "}{.spec.replicas}{"/"}{.status.updatedReplicas}{"/"}{.status.readyReplicas}{"  rook-version="}{.metadata.labels.rook-version}{"\n"}{end}'
#+end_src

** Deploy(Upgrade)

#+begin_src bash
helm upgrade rook-ceph rook-release/rook-ceph \
  --namespace rook-ceph \
  --version v1.x.y
#+end_src

#+begin_src bash
helm upgrade rook-ceph-cluster rook-release/rook-ceph-cluster \
  --namespace rook-ceph \
  --set operatorNamespace=rook-ceph \
  -f cluster-values.yaml \
  --version v1.x.y
#+end_src

または (valuesを再利用する場合)

#+begin_src bash
helm upgrade rook-ceph-cluster rook-release/rook-ceph-cluster \
  --namespace rook-ceph \
  --set operatorNamespace=rook-ceph \
  --reuse-values
  --version v1.x.y
#+end_src

** cluster-values.yaml

#+begin_src yaml
cephClusterSpec:
  storage:
    nodes:
    - devices:
      - name: /dev/nvme0n1p3
      name: k8s-worker1
    - devices:
      - name: /dev/nvme0n1p3
      name: k8s-worker2
    - devices:
      - name: /dev/nvme0n1p3
      name: k8s-worker3
    useAllDevices: false
    useAllNodes: false
#+end_src

** Helm repo の更新

#+begin_src shell
helm repo update
#+end_src

** チャートのバージョンを確認

利用可能な v1.x チャートのバージョンを確認します。

#+begin_src shell
curl -s https://charts.rook.io/release/index.yaml | grep "version: v1.x"
#+end_src

最新の v1.x.y をデプロイします。
