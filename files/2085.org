#+TITLE: Raspberry Pi (bookworm:12) に Emacs 30.2 をインストール

* はじめに

Raspberry Pi (bookworm:12) に Emacs 30.2 をインストールしました。

インストール環境は下記です。

#+begin_src bash
wurly@rp4:~ $ lsb_release -a
No LSB modules are available.
Distributor ID: Debian
Description:    Debian GNU/Linux 12 (bookworm)
Release:        12
Codename:       bookworm
#+end_src

下記のコンフィギュレーションとなります。

* 手順

** ソースコードの取得

#+begin_src bash
$ git clone --depth 1 --branch emacs-30.2 git://git.sv.gnu.org/emacs.git emacs-30.2
#+end_src

** 依存するパッケージのインストール

#+begin_src bash
sudo apt install texinfo autoconf
#+end_src

#+begin_src bash
$ sudo apt install libgccjit-12-dev
#+end_src

#+begin_src bash
sudo apt-get -y install libncurses-dev \
                        libgtk-3-dev \
                        libjpeg-dev \
                        libgif-dev \
                        libtiff-dev \
                        gnutls-dev \
                        librsvg2-dev \
                        libpng-dev \
                        libxpm-dev \
                        libtree-sitter-dev
#+end_src

#+begin_src bash
sudo apt install cmake libtool-bin markdown
#+end_src

** Makefileの生成

#+begin_src bash
cd emacs-30.2
./autogen.sh
./configure --with-native-compilation --without-pop --with-mailutils --with-tree-sitter
#+end_src --with-imagemagick
