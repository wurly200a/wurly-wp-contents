* 概要

* clientのアップグレード

#+begin_src bash
echo "deb [signed-by=/etc/apt/keyrings/kubernetes-apt-keyring.gpg] https://pkgs.k8s.io/core:/stable:/v1.33/deb/ /" \
  | sudo tee /etc/apt/sources.list.d/kubernetes.list
sudo apt update
sudo apt upgrade kubectl
#+end_src

* アップグレード from v1.32.13 to v1.33.9

** 0) まず ctrl1 で etcd snapshot

※rootシェルに入るので、exitの実行を忘れないようにすること

#+begin_src bash
sudo -i
ETCDCTL_API=3 etcdctl snapshot save /root/etcd-$(date +%F-%H%M)-pre133.db \
  --endpoints=https://127.0.0.1:2379 \
  --cacert=/etc/kubernetes/pki/etcd/ca.crt \
  --cert=/etc/kubernetes/pki/etcd/server.crt \
  --key=/etc/kubernetes/pki/etcd/server.key
exit
#+end_src

** 0) 途中監視(Ceph)

別ターミナル：

#+begin_src 
watch -n 2 'kubectl get nodes; echo; kubectl -n rook-ceph get pods -o wide | egrep "mon|mgr|osd|mds|rgw|operator"'
#+end_src

** 1) ctrl1（最初の1台だけ “apply”）

#+begin_src bash
# kubeadm が 1.33.9 になっていることを確認

# イメージpull
sudo kubeadm config images pull

# アップグレード本番
sudo kubeadm upgrade apply v1.33.9

# バージョン文字列確認
apt-cache madison kubelet | grep 1.33.9
apt-cache madison kubectl | grep 1.33.9

# kubelet/kubectl を 1.33.9 に
sudo apt-get update
sudo apt-get install -y kubelet=1.33.9-1.1 kubectl=1.33.9-1.1
sudo systemctl daemon-reload
sudo systemctl restart kubelet
#+end_src

** 2) ctrl2 / ctrl3（各ノード、順番に）

※ここは apply じゃなくて upgrade node

#+begin_src bash
sudo apt-get update

sudo kubeadm upgrade node
sudo apt-get install -y kubelet=1.33.9-1.1 kubectl=1.33.9-1.1
sudo systemctl daemon-reload
sudo systemctl restart kubelet
#+end_src
